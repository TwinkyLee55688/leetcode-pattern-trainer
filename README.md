# LeetCode Pattern Trainer

A static LeetCode pattern training page for drilling LC150-style problems, hints, solution reading, follow-ups, and Google-style Q&A practice.

## Open Locally

Open `trainer.html` directly in a browser, or serve the folder with any static file server.

## GitHub Pages

After pushing this repository to GitHub, enable GitHub Pages from:

`Settings -> Pages -> Deploy from a branch -> main -> /root`

Then open:

`https://<your-github-name>.github.io/<repo-name>/trainer.html`

## Notes

- Progress is stored in browser `localStorage`, so each device keeps its own answer history.
- The app is static HTML/CSS/JavaScript and does not require a backend.
- The Chrome extension files are kept for local extension usage, but GitHub Pages should open `trainer.html`.
